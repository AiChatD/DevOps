# CICD流水线

## 实现功能
### 1、代码自动构建(我这里做的手动触发，你可以调整为自动触发构建)
### 2、流转审批节点
### 3、触发钉钉或企业微信或其他平台机器人发布审批信息
### 4、审批人登入Jenkins填写随机Token确认审批发布
### 5、Token认证成功,即可发布线上K8s集群
